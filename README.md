# Customer Segmentation using K-Means Clustering

## Project Overview

This project analyzes mall customer data using Python to identify distinct customer segments based on income, spending behavior, and demographics. The goal is to understand customer patterns and support data-driven marketing decisions.

## Objective

* Identify different customer groups based on behavior
* Analyze relationships between income, age, and spending score
* Apply clustering techniques to segment customers
* Support business strategies using data insights

## Tools and Technologies

* Python (Pandas, Matplotlib, Seaborn)
* Scikit-learn (K-Means Clustering)

## Analysis Performed

* Univariate Analysis (distribution plots, boxplots)
* Bivariate Analysis (scatter plots, pairplots)
* Multivariate Analysis (correlation heatmap)

## Model

* K-Means Clustering
* Elbow Method for optimal cluster selection

## Key Insights

* High income and high spending customers represent premium segments
* Low income and low spending customers represent budget segments
* Some high income customers have low spending, indicating potential target groups
* Spending behavior varies across different age groups
